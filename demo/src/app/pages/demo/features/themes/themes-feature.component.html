Theme Characteristics

Default

Current styling with sharp edges, clean borders, and professional appearance.

Material Design

Rounded corners (4px), elevation shadows, 8dp grid system, and Material Design colors.

Bootstrap

Bootstrap 5 styling with rounded corners (6px), standard spacing, and Bootstrap colors.

Ant Design

Sharp edges (0px), high contrast, compact spacing, and Ant Design color scheme.

Ant Design Alt

Same as Ant Design but with row borders only (no column borders).

Global Setup

To use the table with its default themes and variables, make sure to import the core styles in your global styles (e.g., styles.scss): 

// Import library variables and core styles
@import 'tangrid/styles/variables';

// If you are using a pre-built theme, you might need to import it as well
// @import 'tangrid/styles/themes/default';

Overriding Themes

You can override any theme by targeting the theme class in your global styles or component styles (using ::ng-deep if needed). The themes use CSS variables which makes them easy to customize. 

.tan-grid--theme-material {{ '{' }}
  --ngs-primary: #ff4081; /* Change primary color */
  --ngs-border-radius: 8px; /* Change border radius */
{{ '}' }}

Available CSS Variables

The following CSS variables are available for customization:

Colors
  • --ngs-primary: Primary brand color (used for active states, links, focus rings)
  • --ngs-primary-hover: Hover state for primary actions
  • --ngs-primary-active: Active state for primary actions
  • --ngs-primary-light: Light shade of primary color (backgrounds)
  • --ngs-secondary: Secondary color (neutral actions)
  • --ngs-text-primary: Main text color
  • --ngs-text-secondary: Secondary text color (headers, hints)
  • --ngs-text-muted: Muted text color (disabled states)
  • --ngs-text-inverse: Inverse text color (text on primary background)
  • --ngs-bg-primary: Primary background color (table body)
  • --ngs-bg-secondary: Secondary background color (headers, striped rows)
  • --ngs-bg-tertiary: Tertiary background color (hover states)
  • --ngs-border-color: Main border color
Typography
  • --ngs-font-family: Font family for the table
  • --ngs-font-size-base: Base font size (14px default)
  • --ngs-font-size-sm: Small font size (13px default)
Layout & Spacing
  • --ngs-border-radius: Border radius for container and inputs
  • --ngs-spacing-xs: Extra small spacing (2px)
  • --ngs-spacing-sm: Small spacing (4px)
  • --ngs-spacing-md: Medium spacing (8px)
  • --ngs-spacing-lg: Large spacing (12px)

Creating a Custom Theme

To create a completely new theme, you can define a class that sets these variables and pass it to the theme input. Note that for custom themes not in the predefined list, you might need to handle the class application manually or extend the types. 

/* global-styles.scss */
.tan-grid--theme-cyberpunk {{ '{' }}
  --ngs-primary: #0ff;
  --ngs-bg-primary: #000;
  --ngs-text-primary: #0f0;
  --ngs-border-radius: 0;
  --ngs-font-family: 'Courier New', monospace;
{{ '}' }}
